# Klinické studie

## Vyhledávání v ClinicalTrials.gov

> *Najdi aktivní klinické studie pro melanom s BRAF mutací.*

```bash
czechmedmcp trial search --condition "melanoma" --status RECRUITING
```

## Filtrování

- **Podmínka**: `--condition "lung cancer"`
- **Stav**: `--status RECRUITING` (RECRUITING, COMPLETED, ACTIVE_NOT_RECRUITING, ...)
- **Fáze**: `--phase "Phase 3"`
- **Lokalita**: `--latitude 50.08 --longitude 14.43 --distance 100` (Praha, 100 km)
