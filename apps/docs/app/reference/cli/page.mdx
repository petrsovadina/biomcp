# CLI příkazy

CzechMedMCP poskytuje CLI přes příkaz `czechmedmcp`.

## Hlavní příkazy

```bash
czechmedmcp run                    # Spuštění MCP serveru
czechmedmcp --help                 # Nápověda
czechmedmcp --version              # Verze
```

## Články

```bash
czechmedmcp article search --gene BRAF --disease melanoma --limit 5
czechmedmcp article get --pmid 12345678
```

## Klinické studie

```bash
czechmedmcp trial search --condition "lung cancer" --status RECRUITING
czechmedmcp trial get --nct-id NCT04236687
```

## Varianty

```bash
czechmedmcp variant search --gene TP53 --significance pathogenic
czechmedmcp variant get --variant-id "chr17:g.7577120G>A"
```

## České nástroje

```bash
# SUKL
czechmedmcp czech sukl search --query "Ibuprofen"
czechmedmcp czech sukl detail --sukl-code "0001234"

# MKN-10
czechmedmcp czech mkn10 search --query "diabetes mellitus"
czechmedmcp czech mkn10 detail --code "E11"

# NRPZS
czechmedmcp czech nrpzs search --city "Praha" --specialty "kardiologie"

# SZV
czechmedmcp czech szv search --query "EKG"
```

## Geny, nemoci, léčiva

```bash
czechmedmcp gene get TP53
czechmedmcp disease get "melanoma"
czechmedmcp drug get "imatinib"
```
