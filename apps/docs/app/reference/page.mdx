# Rychlá reference

## Instalace

```bash
uv tool install czechmedmcp
# nebo
pip install czechmedmcp
```

## Spuštění

```bash
czechmedmcp run           # MCP server (stdio)
czechmedmcp --help        # Nápověda
```

## Claude Desktop konfigurace

```json
{
  "mcpServers": {
    "czechmedmcp": {
      "command": "uv",
      "args": ["run", "--with", "czechmedmcp", "czechmedmcp", "run"]
    }
  }
}
```

## Proměnné prostředí

| Proměnná | Povinná | Popis |
|----------|---------|-------|
| `NCBI_API_KEY` | Ne | PubMed rate limit |
| `NCI_API_KEY` | Ne | NCI klinické studie |
| `ONCOKB_API_KEY` | Ne | OncoKB anotace |
| `GOOGLE_CLOUD_PROJECT` | Ne | BigQuery logging |

## Cache

- Umístění: `~/.cache/czechmedmcp/http_cache`
- Výchozí TTL: 1 týden
- Vymazání: `rm -rf ~/.cache/czechmedmcp/`
