# Lokální vývoj

## Prerekvizity

- Python 3.10+
- [uv](https://docs.astral.sh/uv/) (doporučeno)
- Git

## Nastavení

```bash
git clone https://github.com/petrsovadina/czechmedmcp.git
cd biomcp
git checkout python-main

# Instalace závislostí
uv sync --all-extras

# Pre-commit hooks
uv run pre-commit install
```

## Spuštění serveru

```bash
# MCP server (stdio)
uv run czechmedmcp run

# MCP Inspector (webové rozhraní pro testování)
make inspector
```

## Užitečné příkazy

```bash
# Spuštění všech testů
uv run python -m pytest -x --ff -n auto --dist loadscope

# Lint
uv run ruff check src tests

# Typová kontrola
uv run mypy

# Formátování
uv run ruff format src tests

# Kompletní kontrola kvality
make check

# Build wheel
make build
```

## Struktura projektu

```
biomcp/
├── src/czechmedmcp/          # Zdrojový kód
│   ├── __init__.py      # Import registrace nástrojů

│   ├── router.py        # Unifikované search/fetch

│   ├── bdd/             # BDD testy
│   └── integration/     # Live API testy

```
