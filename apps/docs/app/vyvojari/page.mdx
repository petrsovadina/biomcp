# Pro vývojáře

Průvodce pro přispěvatele a vývojáře CzechMedMCP.

## Repozitář

- **GitHub**: [github.com/petrsovadina/czechmedmcp](https://github.com/petrsovadina/czechmedmcp)
- **Branch**: `main`
- **Licence**: MIT

## Rychlý start pro vývojáře

```bash
git clone https://github.com/petrsovadina/czechmedmcp.git
cd CzechMedMCP
uv sync --all-extras
uv run pre-commit install
uv run python -m pytest -x --ff -n auto
```
