# Styl kódu

## Pravidla

- **Python 3.10+**
- Délka řádku: **79 znaků** (ruff)
- Union syntaxe: `X | Y` (ne `Optional[X]`)
- Formáter: `ruff format`
- Linter: `ruff check`
- Typová kontrola: `mypy`

## Tool funkce

```python
@mcp_app.tool()
async def my_tool(
    param: Annotated[str, Field(description="Popis parametru")],
) -> str:
    """Stručný popis nástroje."""
    result = await _private_impl(param)
    return render.to_markdown(result)
```

- Vždy vracejí `str`
- Parametry s `Annotated[type, Field(description="...")]`
- Volitelný `@track_performance("czechmedmcp.tool_name")`
