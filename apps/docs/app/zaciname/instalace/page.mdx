# Instalace

## Doporučený způsob (uv)

[uv](https://docs.astral.sh/uv/) je moderní Python package manager — rychlý a spolehlivý.

```bash
# Instalace uv (pokud nemáte)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Instalace CzechMedMCP jako CLI nástroj
uv tool install git+https://github.com/petrsovadina/CzechMedMCP.git

# Ověření
czechmedmcp --help
```

> **Poznámka**: Balíček zatím není na PyPI — instalace probíhá přímo z Git repozitáře.

## Pro vývojáře (editovatelná instalace)

```bash
git clone https://github.com/petrsovadina/czechmedmcp.git
cd CzechMedMCP
uv sync --all-extras
uv run pre-commit install
```

## Ověření instalace

```bash
# Spuštění serveru
czechmedmcp run

# Nebo přes uv
uv run czechmedmcp run
```

Server se spustí na stdio transportu a čeká na MCP klienta.

## Docker

```bash
docker compose up -d
```

Viz `docker-compose.yml` v repozitáři pro konfiguraci.
