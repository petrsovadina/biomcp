# První dotaz

## Přes Claude Desktop

Otevřete Claude Desktop a zeptejte se:

> *Vyhledej lék Ibuprofen v SUKL registru a ukaž mi jeho dostupnost.*

Claude automaticky použije nástroje `czechmed_search_medicine` a `czechmed_check_availability`.

## Přes CLI

```bash
# Vyhledání léku
czechmedmcp czech sukl search --query "Ibuprofen"

# Vyhledání diagnózy
czechmedmcp czech mkn10 search --query "diabetes mellitus"

# Vyhledání poskytovatele
czechmedmcp czech nrpzs search --city "Praha" --specialty "kardiologie"

# Globální vyhledávání
czechmedmcp article search --gene BRAF --disease melanoma
czechmedmcp trial search --condition "lung cancer" --status RECRUITING
```

## Ukázkové scénáře

### Scénář 1: Lékař hledá informace o léku

> *Najdi v SUKL registru přípravky s účinnou látkou metformin. U prvního výsledku zobraz SPC sekci „Dávkování" a zkontroluj dostupnost.*

### Scénář 2: Diagnostická podpora

> *Pacient má bolest na hrudi, dušnost a otoky dolních končetin. Jaké diagnózy MKN-10 by mohly odpovídat? Najdi ke každé diagnóze relevantní články v PubMed.*

### Scénář 3: Odeslání pacienta

> *Potřebuji odeslat pacienta s diagnózou I25.1 ke kardiologovi v Brně. Najdi vhodné poskytovatele.*
